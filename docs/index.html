Interactive Diagrams Library (IDL)

Librería JavaScript nativa para diagramas interactivos con renderizado SVG y Canvas

SVG + Canvas 148 clases 5 layouts 14 herramientas v0.3.0

Renderizado Dual

Soporte nativo para SVG y Canvas con switch automático según viewport.

Layouts Avanzados

Árbol, fuerza dirigida, circular, grid y layered digraph (Sugiyama).

Interacción Completa

Drag, resize, rotate, linking, panning, zoom y edición de texto in-place.

Modelos Flexibles

GraphLinksModel para grafos y TreeModel para jerarquías padre-hijo.

Bindings Declarativos

Sincronización bidireccional modelo ↔ vista sin código repetitivo.

0 Dependencias

JavaScript puro. Sin dependencias externas. Compatible con cualquier framework.
