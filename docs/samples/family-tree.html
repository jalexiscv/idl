Árbol Genealógico — Nobleza Británica

Rey/Reina Príncipe/Princesa Masculino Femenino

Sobre este ejemplo

Árbol genealógico de la nobleza británica desde Jorge V. Cada nodo muestra nombre, fechas de nacimiento/defunción y género. El color del borde indica el estatus real (dorado = rey/reina, azul = príncipe/princesa, verde = civil). La insignia de género usa azul para masculino y rosa para femenino.

diagram.layout = new idl.TreeLayout({
  angle: 90,
  nodeSpacing: 15,
  layerSpacing: 60,
  arrangement: idl.TreeLayout.ArrangementVertical
});

diagram.model = new idl.TreeModel({
  nodeKeyProperty: "name",
  nodeParentProperty: "parent",
  nodeDataArray: [ /* datos genealógicos */ ]
});
