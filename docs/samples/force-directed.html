Layout de Fuerza Dirigida

About this sample

Simulación de fuerzas con ForceDirectedLayout: los nodos se repelen entre sí y los enlaces actúan como resortes. Útil para visualizar grafos de redes sociales, coautoría, dependencias, etc.

diagram.layout = new idl.ForceDirectedLayout({
  maxIterations: 300,
  defaultSpringStiffness: 0.03,
  defaultSpringLength: 80,
  defaultElectricalCharge: 150,
});
