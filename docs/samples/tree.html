Layout de Árbol

About this sample

Estructura jerárquica de árbol usando TreeModel y TreeLayout. Los nodos se posicionan automáticamente en capas. El layout se re-ejecuta al hacer cambios.

diagram.layout = new idl.TreeLayout({
  angle: 90,
  layerSpacing: 50,
  nodeSpacing: 20,
  arrangement: idl.TreeLayout.ArrangementVertical
});

diagram.model = new idl.TreeModel([
  { key: 1, text: "CEO" },
  { key: 2, text: "VP Sales", parent: 1 },
  { key: 3, text: "VP Engineering", parent: 1 },
  { key: 4, text: "Sales Manager", parent: 2 },
  { key: 5, text: "Sales Rep A", parent: 4 },
  { key: 6, text: "Sales Rep B", parent: 4 },
  { key: 7, text: "Dev Manager", parent: 3 },
  { key: 8, text: "Frontend Lead", parent: 7 },
  { key: 9, text: "Backend Lead", parent: 7 },
]);
