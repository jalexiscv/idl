Diagrama Entidad-Relación (ER)

Clave Texto Numérico Binario

Sobre este ejemplo

Diagrama entidad-relación con entidades como tablas de base de datos, atributos codificados por color semántico y relaciones con cardinalidad. Los nodos usan Panel.Table para organizar los atributos en filas. Las relaciones muestran la cardinalidad en cada extremo.

diagram.layout = new idl.ForceDirectedLayout({
  defaultSpringLength: 200,
  defaultElectricalCharge: 300,
});

// Atributos codificados por tipo:
//   Diamante púrpura = Clave primaria/foránea
//   Rectángulo azul  = Texto/descripción
//   Círculo verde    = Numérico/monetario
//   Triángulo rojo   = Binario/imagen
