Basic Flowchart

About this sample

Un diagrama de flujo simple con nodos personalizados (RoundedRectangle + TextBlock) y enlaces dirigidos. Incluye animación de layout TreeLayout al cargar.

diagram.model = new idl.GraphLinksModel({
  nodeDataArray: [
    { key: 1, text: "Start" },
    { key: 2, text: "Process" },
    { key: 3, text: "Decision" },
    { key: 4, text: "End" },
    { key: 5, text: "Alternative" },
  ],
  linkDataArray: [
    { from: 1, to: 2 },
    { from: 2, to: 3 },
    { from: 3, to: 4 },
    { from: 3, to: 5 },
  ]
});
