Circular Layout

About this sample

Los nodos se disponen en círculo con CircularLayout. Los enlaces entre nodos adyacentes y cruzados muestran diferentes patrones de conexión. Ideal para visualizar topologías de red, ciclos y relaciones circulares.

diagram.layout = new idl.CircularLayout({
  radius: 180,
  spacing: 20,
  nodeDiameterFormula: idl.CircularLayout.Circular,
  sorting: idl.CircularLayout.Ascending
});
